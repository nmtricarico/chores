Data Tools

Settings & Maintenance

These tools are for backup, transfer, troubleshooting, and starting fresh. Your daily chore dashboard stays clean and touch-focused.

Reset Progress

This clears all saved chore progress and sets every chore back to due for the current day, week, or month.

Export Progress

Copy this JSON text or download it as a backup file. This can be imported later on the same iPad or another browser.

Import Progress

Paste previously exported JSON here, then import it. Importing replaces the currently saved progress.

Debug State

This shows the raw saved state that the app is using internally. It is helpful when testing resets and imports.
